O projeto utilizará o princípio da autonomia, pois, como a plataforma depende dos dados do usuário para correlacionar onde ele está olhando, será solicitado o consentimento prévio antes da realização dos testes. Para isso, será apresentado um Termo de Consentimento Livre e Esclarecido (TCLE), garantindo que o participante autorize o uso de seus dados de forma consciente.
Além disso, será aplicado o princípio da beneficência, uma vez que a plataforma tem como objetivo auxiliar no direcionamento adequado de propagandas, promovendo maior assertividade tanto para o público que as recebe quanto para os anunciantes que investem na divulgação de seus produtos.
